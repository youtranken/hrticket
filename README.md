# HRIS / C&B Ticket Management

Internal ticket system that turns two HR/C&B Gmail mailboxes into managed tickets. On-premise, two isolated projects (`hris`, `cnb`), ~40 users.

- **Planning docs:** `../_bmad-output/planning-artifacts/` (PRD, architecture, epics & stories).
- **AI dev guide:** [`CLAUDE.md`](./CLAUDE.md) — read first each session.
- **Full conventions:** [`CONVENTIONS.md`](./CONVENTIONS.md).

## Stack
Node 24 · NestJS 11 · React 19 · Vite 8 · TypeScript 5 · PostgreSQL 18.4 · Drizzle ORM · pnpm workspaces · Ant Design · Docker Compose.
